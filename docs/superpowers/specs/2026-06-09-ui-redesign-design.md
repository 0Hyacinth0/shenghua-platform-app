# 盛桦学堂 UI 重设计方案

## 概述

对 `client/` 前端项目进行全面 UI 重设计，统一视觉风格，提升整体精致度和排版布局质量。

**目标**：将当前粗糙的橙红色 UI 升级为「现代清新」风格 — 深色导航 + 紫色强调 + 柔和色块卡片。

**范围**：所有页面统一重设计（首页、学习中心、社区、个人中心、课程详情、课程播放、课程结算、我的课程、订单、地址、优惠券、签到、拼团、消息等）。

## 设计方向

**风格**：Modern Fresh（现代清新）
- 深色导航栏 (#1A1A2E) 作为视觉锚点
- 紫色 (#8B5CF6) 作为强调色
- 柔和色块卡片（薰衣草、暖橙、薄荷绿、暖黄、天蓝、珊瑚红）
- 大圆角、轻阴影、充足留白
- 药丸形按钮、圆形图标按钮

**参考**：Figma / Notion 的简洁现代感，结合学习平台的温暖亲和力。

**不采用**：Liquid Glass 效果（用户确认不需要）。

## 设计系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色·深色 | #1A1A2E | 顶部导航、重要文字、主按钮 |
| 强调色·紫色 | #8B5CF6 | 标签、进度条、选中态、次要按钮 |
| 页面背景 | #F7F8FA | 页面底色 |
| 卡片背景 | #FFFFFF | 卡片、弹窗 |
| 薰衣草 | #F5F3FF | 编程/技术分类背景 |
| 暖橙 | #FFF4EE | 设计/创作分类背景 |
| 薄荷绿 | #F0FDF4 | 语言/成功状态背景 |
| 暖黄 | #FEF3C7 | AI/警告状态背景 |
| 天蓝 | #E0F2FE | 信息状态背景 |
| 珊瑚红 | #FEE2E2 | 危险/删除状态背景 |
| 文字主色 | #1A1A2E | 标题、正文 |
| 文字辅色 | #64748B | 辅助信息 |
| 文字提示 | #94A3B8 | 占位符、时间戳 |
| 成功 | #22C55E | 免费标签、成功提示 |
| 警告 | #F59E0B | 价格、警告提示 |
| 危险 | #EF4444 | 删除、错误提示 |
| 信息 | #0EA5E9 | 链接、提示 |

### 文字层次

| 级别 | 大小 | 字重 | 用途 |
|------|------|------|------|
| 页面标题 | 22px | Bold (700) | 页面顶部标题 |
| 模块标题 | 17px | Semibold (600) | section 标题 |
| 卡片标题 | 15px | Semibold (600) | 卡片内标题 |
| 正文 | 14px | Regular (400) | 正文内容 |
| 辅助文字 | 13px | Regular (400) | 次要信息，灰色 |
| 标签文字 | 11px | Regular (400) | 标签、角标 |

### 间距系统

| 间距 | 值 | 用途 |
|------|------|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 元素内间距 |
| md | 12px | 卡片内间距 |
| lg | 16px | 页面边距 |
| xl | 20px | 模块间距 |

### 圆角系统

| 圆角 | 值 | 用途 |
|------|------|------|
| sm | 8px | 小标签、进度条 |
| md | 12px | 图标容器、输入框 |
| lg | 14px | 卡片、色块 |
| xl | 16px | 大卡片、弹窗 |
| pill | 20px | 按钮、标签 |
| full | 50% | 头像、圆形按钮 |

### 阴影

| 级别 | 值 | 用途 |
|------|------|------|
| sm | 0 1px 3px rgba(0,0,0,0.04) | 卡片默认 |
| md | 0 2px 8px rgba(0,0,0,0.04) | 悬浮态、统计卡 |

### 组件样式

**卡片**：
- 背景 #FFFFFF，圆角 14px，内间距 14-16px
- 阴影 sm，无边框
- 深色卡片（统计区等）：背景 #1A1A2E，圆角 16px

**按钮**：
- 主按钮：背景 #1A1A2E，文字 #fff，圆角 pill，内间距 12px 24px
- 次要按钮：背景 #F5F3FF，文字 #8B5CF6，圆角 pill
- 文字按钮：无背景，文字 #8B5CF6

**标签**：
- 圆角 pill，内间距 3px 10px，字号 11px
- 分类标签用对应色块背景 + 深色文字
- 状态标签用功能色

**输入框**：
- 背景 #F7F8FA，圆角 12px，内间距 10px 14px
- 边框 1px solid #E2E8F0

**导航栏**：
- 背景 #1A1A2E，padding-top 44px（状态栏）
- 搜索框：rgba(255,255,255,0.1) 背景，圆角 pill

**图标**：
- 统一使用 @iconify/vue 的 Icon 组件
- 禁止使用 emoji

## 页面设计

### 首页 (pages/home/index)

**结构**：
1. 深色导航栏 — 搜索框 + 消息按钮
2. Banner 轮播 — 深色卡片 + 紫色点缀，简洁文案
3. 课程分类 — 2 列大色块卡片网格（替代当前 4 列小图标）
4. 热门课程 — 竖向列表卡片（替代当前横向滚动）
5. 免费公开课 — 竖向列表卡片
6. 精选好课 — 竖向列表卡片

**改动重点**：
- 顶部：橙色渐变 → 深色背景
- 分类：4 列小图标 → 2 列大色块卡片，显示课程数量
- 课程：横向滚动 → 竖向列表，信息更丰富（封面+标题+讲师+价格+学习人数）
- 保留频道 Tab（推荐/热门/新课/直播）— 放在热门课程区域上方
- 快捷入口移至学习中心
- 明星讲师移至学习中心

### 学习中心 (pages/learn/index)

**结构**：
1. 深色导航栏 — 标题 + 设置按钮
2. 深色统计卡 — 已购课程 / 学习时长 / 获得证书
3. 快捷入口 — 4-5 列图标网格（限时特惠、优惠券、签到、拼团、免费课，从首页移入）
4. 功能入口 — 4 列图标网格（我的课程、学习计划、我的证书、我的收藏）
5. 明星讲师 — 横向滚动卡片（从首页移入）
6. 继续学习 — 横向卡片（封面+标题+进度条）
7. 我的课程 — Tab 切换（在学/已完成/已购）+ 列表卡片

### 社区 (pages/community/index)

**结构**：
1. 深色导航栏 — 标题 + 搜索 + 发帖按钮
2. 话题标签 — 横向滚动标签（全部/学习心得/求助/资源分享）
3. 帖子列表 — 卡片（头像+用户名+时间+标签+内容+图片+互动栏）

**改动重点**：
- 导航栏改为深色背景
- 话题标签选中态改为 #1A1A2E 背景
- 帖子卡片圆角加大到 14px
- 互动栏图标统一用 @iconify/vue

### 个人中心 (pages/profile/index)

**结构**：
1. 深色头像区 — 头像 + 用户名 + 等级 + 编辑资料按钮
2. 统计卡 — 课程/证书/收藏/连续签到
3. 菜单列表 — 我的订单/优惠券/收货地址/设置

### 课程详情 (pages/course/detail)

**结构**：
1. 渐变封面 — 播放按钮 + 返回按钮
2. 标签 — 分类 + 免费/付费
3. 课程信息 — 标题 + 讲师 + 课时 + 学习人数
4. 讲师卡片 — 头像 + 名称 + 简介
5. 课程目录 — 序号 + 标题 + 时长
6. 底部 CTA — 主按钮 + 收藏按钮

### 其他页面

所有页面统一应用以下规则：
- 顶部导航栏使用深色 (#1A1A2E) 背景
- 卡片使用白色背景 + 14px 圆角
- 标签使用药丸形状 + 对应色块
- 按钮使用药丸形状
- 图标统一使用 @iconify/vue
- 禁止使用 emoji

## 约束

1. **禁止 emoji** — 所有图标使用 @iconify/vue 的 Icon 组件
2. **统一导航栏** — 所有页面顶部使用深色 (#1A1A2E) 背景
3. **统一卡片** — 白色背景、14px 圆角、sm 阴影
4. **统一按钮** — 药丸形状
5. **统一标签** — 药丸形状 + 对应色块
6. **保持现有功能** — 只改视觉，不改交互逻辑
7. **保持跨端兼容** — CSS 属性需兼容小程序/App
