- **最后更新**：2026-05-27

## 接口列表

### 1.1 发送短信验证码

POST /api/app/edu/v1/auth/sms-code

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| phone | String | 是 | 手机号，11 位 |

**请求示例**

```json
{
  "phone": "13200006054"
}
```

**响应示例**

```json
{
  "success": true,
  "code": 200,
  "message": "验证码已发送",
  "result": true,
  "timestamp": 1748332800000
}
```

**业务约束**

- 同一手机号 60 秒内不可重复发送。
- 同一 IP 每分钟不超过 10 次。
- 验证码有效期 5 分钟。
- result 为 `true` 表示发送成功。

### 1.2 手机号验证码登录

POST /api/app/edu/v1/auth/login-by-sms

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| phone | String | 是 | 手机号，11 位 |
| smsCode | String | 是 | 短信验证码，6 位 |
| privacyAgreementVersion | String | 是 | 用户同意的隐私协议版本号 |

**请求示例**

```json
{
  "phone": "13200006054",
  "smsCode": "123456",
  "privacyAgreementVersion": "2026-05-01"
}
```

**响应示例**

```json
{
  "success": true,
  "code": 200,
  "message": "登录成功",
  "result": {
    "token": "eyJhbGciOiJIUzI1NiJ9...",            // String
    "user": {
      "id": 1,                                      // Long
      "nickname": "盛桦用户",                         // String | 可空
      "avatar": "https://cdn.example.com/avatar.png", // String | 可空
      "phoneMasked": "132****6054"                   // String
    }
  },
  "timestamp": 1748332800000
}
```

**响应字段说明**

| 字段 | 类型 | 可空 | 说明 |
| --- | --- | --- | --- |
| token | String | 否 | JWT Token，后续请求通过 `Authorization: Bearer <token>` 传递 |
| user.id | Long | 否 | 用户唯一 ID |
| user.nickname | String | 是 | 昵称，新用户可能为空 |
| user.avatar | String | 是 | 头像 URL，新用户可能为空 |
| user.phoneMasked | String | 否 | 脱敏手机号，格式 `132****6054` |

**业务约束**

- 验证码错误返回 `{"success": false, "code": 422, "message": "验证码错误或已过期"}`。
- 首次登录自动创建用户（手机号作为唯一标识），nickname 默认为设置一个随机名字。
- 隐私协议版本号必须记录到用户表或独立的协议记录表。
- Token 有效期建议 7 天，过期返回 401。
