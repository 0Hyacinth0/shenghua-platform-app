## 接口列表

### 2.1 课程分类列表

GET /api/app/edu/v1/course-categories

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 无 | — | — | 返回所有已启用分类 |

**响应示例**

```json
{
  "success": true,
  "code": 200,
  "message": "操作成功",
  "result": [
    {
      "id": 1,                                     // Long
      "name": "考研",                                // String
      "icon": "https://cdn.example.com/kaoyan.png",  // String | 可空
      "sortNo": 1,                                  // Integer
      "courseCount": 12                             // Integer
    }
  ],
  "timestamp": 1748332800000
}
```

**响应字段说明**

| 字段 | 类型 | 可空 | 说明 |
| --- | --- | --- | --- |
| id | Long | 否 | 分类 ID |
| name | String | 否 | 分类名称 |
| icon | String | 是 | 分类图标 URL |
| sortNo | Integer | 否 | 排序号 |
| courseCount | Integer | 否 | 该分类下已发布课程数量 |

**业务约束**

- 只返回 `status = 'enabled'` 的分类。
- 按 `sort_no` 升序排列。
- `courseCount` 只统计已发布课程。

### 2.2 课程列表

GET /api/app/edu/v1/courses

**请求参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| categoryId | Long | 否 | 分类 ID，不传返回所有分类 |
| keyword | String | 否 | 搜索关键词，匹配课程标题 |
| pageNo | Integer | 是 | 页码，从 1 开始 |
| pageSize | Integer | 是 | 每页条数，默认 10 |

**响应示例**

```json
{
  "success": true,
  "code": 200,
  "message": "操作成功",
  "result": {
    "records": [
      {
        "id": 1,                                      // Long
        "title": "3天拿捏事业编做题技巧课",               // String
        "coverUrl": "https://cdn.example.com/c1.png",  // String
        "teacherName": "付晓鹏",                        // String
        "lessonCount": 3,                              // Integer
        "studentCount": 1230,                          // Integer
        "price": 0,                                    // BigDecimal | 0=免费
        "tags": ["热门", "新上"]                          // List<String> | 可空
      }
    ],
    "total": 50,       // Long
    "size": 10,        // Integer
    "current": 1,      // Long
    "pages": 5         // Long
  },
  "timestamp": 1748332800000
}
```

**响应字段说明**

| 字段 | 类型 | 可空 | 说明 |
| --- | --- | --- | --- |
| id | Long | 否 | 课程 ID |
| title | String | 否 | 课程标题 |
| coverUrl | String | 否 | 封面图 URL |
| teacherName | String | 否 | 主讲老师姓名 |
| lessonCount | Integer | 否 | 课节总数 |
| studentCount | Integer | 否 | 学员数 |
| price | BigDecimal | 否 | 价格，0 表示免费 |
| tags | List\<String\> | 是 | 标签列表 |

**分页字段**

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| records | Array | 当前页记录 |
| total | Long | 总记录数 |
| size | Integer | 每页条数 |
| current | Long | 当前页码 |
| pages | Long | 总页数 |

**业务约束**

- 只返回 `status = 'published'` 的课程。
- `keyword` 对 `title` 做模糊匹配。
- 按发布时间倒序排列。

### 2.3 课程详情

GET /api/app/edu/v1/courses/{courseId}

**路径参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| courseId | Long | 是 | 课程 ID |

**响应示例**

```json
{
  "success": true,
  "code": 200,
  "message": "操作成功",
  "result": {
    "id": 1,                                      // Long
    "title": "3天拿捏事业编做题技巧课",               // String
    "subtitle": "从零基础到高分突破",                 // String | 可空
    "intro": "本课程针对事业编考试...",              // String
    "coverUrl": "https://cdn.example.com/c1.png",  // String
    "price": 0,                                    // BigDecimal
    "lessonCount": 3,                              // Integer
    "totalDuration": 10800,                        // Integer | 秒
    "studentCount": 1230,                          // Integer
    "accessType": "login_required",                // String | public / login_required / enrolled
    "teacher": {
      "id": 1,                                     // Long
      "name": "付晓鹏",                             // String
      "avatar": "https://cdn.example.com/t1.png",  // String | 可空
      "title": "资深讲师",                           // String | 可空
      "intro": "10年教学经验"                        // String | 可空
    },
    "lessons": [
      {
        "id": 1,                                   // Long
        "title": "拿捏核心技法",                     // String
        "duration": 3600,                          // Integer | 秒
        "isTrial": true,                           // Boolean
        "sortNo": 1                                // Integer
      },
      {
        "id": 2,
        "title": "高分答题模板",
        "duration": 2700,
        "isTrial": false,
        "sortNo": 2
      }
    ]
  },
  "timestamp": 1748332800000
}
```

**响应字段说明**

| 字段 | 类型 | 可空 | 说明 |
| --- | --- | --- | --- |
| id | Long | 否 | 课程 ID |
| title | String | 否 | 课程标题 |
| subtitle | String | 是 | 副标题 |
| intro | String | 否 | 课程简介 |
| coverUrl | String | 否 | 封面图 URL |
| price | BigDecimal | 否 | 价格 |
| lessonCount | Integer | 否 | 课节总数 |
| totalDuration | Integer | 否 | 总时长（秒） |
| studentCount | Integer | 否 | 学员数 |
| accessType | String | 否 | public / login_required / enrolled |
| teacher.id | Long | 否 | 讲师 ID |
| teacher.name | String | 否 | 讲师姓名 |
| teacher.avatar | String | 是 | 讲师头像 |
| teacher.title | String | 是 | 讲师头衔 |
| teacher.intro | String | 是 | 讲师简介 |
| lessons[].id | Long | 否 | 课节 ID |
| lessons[].title | String | 否 | 课节标题 |
| lessons[].duration | Integer | 否 | 课节时长（秒） |
| lessons[].isTrial | Boolean | 否 | 是否试看 |
| lessons[].sortNo | Integer | 否 | 排序号 |

**业务约束**

- 课程必须已发布才能返回。
- `lessons` 只返回 `status = 'published'` 的课节，按 `sort_no` 升序。
- 当 `accessType = 'public'` 且用户未登录时，`lessons` 仍返回（允许未登录浏览）。
