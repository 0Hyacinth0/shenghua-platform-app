## 接口列表

### 3.1 获取课节播放信息

GET /api/app/edu/v1/lessons/{lessonId}/play

**路径参数**

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| lessonId | Long | 是 | 课节 ID |

**响应示例**

```json
{
  "success": true,
  "code": 200,
  "message": "操作成功",
  "result": {
    "lessonId": 1,                                                     // Long
    "courseId": 1,                                                     // Long
    "title": "拿捏核心技法",                                             // String
    "coverUrl": "https://cdn.example.com/video-cover.png",             // String | 可空
    "duration": 3600,                                                  // Integer | 秒
    "playAuth": {
      "type": "url",                                                   // String | url / vod_token
      "url": "https://vod.example.com/play/abc123.m3u8",              // String | type=url 时返回
      "expireAt": "2026-05-28 23:59:59"                                // String | 可空
    },
    "lastPosition": 120                                                // Integer | 上次播放位置（秒），0 表示新开始
  },
  "timestamp": 1748332800000
}
```

**响应字段说明**

| 字段 | 类型 | 可空 | 说明 |
| --- | --- | --- | --- |
| lessonId | Long | 否 | 课节 ID |
| courseId | Long | 否 | 所属课程 ID |
| title | String | 否 | 课节标题 |
| coverUrl | String | 是 | 视频封面图 |
| duration | Integer | 否 | 视频时长（秒） |
| playAuth.type | String | 否 | `url` 直接播放地址 / `vod_token` 云点播凭证 |
| playAuth.url | String | 是 | 播放地址，type=url 时返回 |
| playAuth.expireAt | String | 是 | 凭证过期时间，格式 `yyyy-MM-dd HH:mm:ss` |
| lastPosition | Integer | 否 | 上次播放位置（秒），新用户为 0 |

**业务约束**

- 必须登录，否则返回 401。
- 课节必须已发布（`status = 'published'`），否则返回 404 或 422。
- 所属课程必须已发布，否则返回 422。
- 权限校验规则：

| 课程 accessType | 条件 | 结果 |
| --- | --- | --- |
| `public` | 无 | 可播放 |
| `login_required` | 已登录 | 可播放 |
| `enrolled` | 已报名或管理员授权 | 可播放 |
| `enrolled` | 未报名 | 返回 403 "请先加入课程" |

- 试看课节（`isTrial = true`）仅可播放前 60 秒，或按业务规则返回裁剪后的播放地址。
- `lastPosition` 从 `edu_learning_progress` 表读取，供前端实现断点续播。

**错误响应示例（未授权）**

```json
{
  "success": false,
  "code": 403,
  "message": "请先加入课程",
  "result": null,
  "timestamp": 1748332800000
}
```

**错误响应示例（课程已下线）**

```json
{
  "success": false,
  "code": 422,
  "message": "课程已下线，无法播放",
  "result": null,
  "timestamp": 1748332800000
}
```
